# 商业模式设计

## 核心原则

代码免费，官方能力收费。

## 免费部分

- 源码
- 基础安装文档
- 社区版功能
- Issue 反馈
- 普通用户自助部署

## 收费部分

### 官方维护订阅

- 官方稳定版
- 官方更新源
- 安全补丁
- 优先修复 Bug
- 兼容性适配
- 私有技术群
- 远程排障
- 升级协助

### 私有部署服务

- 服务器环境初始化
- 面板安装
- Agent 接入
- 域名证书配置
- 备份策略
- 监控告警
- 数据迁移

### 企业支持

- SLA 响应
- 定制开发
- 品牌配置
- 代理商系统
- 工单系统
- 多租户隔离
- 专属培训

## 价格形态草案

- 社区版：免费
- 官方维护订阅：按年收费
- 私有部署：一次性部署费 + 年维护费
- 企业定制：单独报价
- 官方认证服务商：年费 + 技术考核
