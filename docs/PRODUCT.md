# ZXY Panel 产品定义

## 一句话定位

ZXY Panel 是一套面向跨境业务、外贸团队、海外社媒运营、AI 工具使用团队和服务器服务商的开源服务器节点管理面板。

## 目标用户

1. 普通技术用户：自用单机面板。
2. 小团队：管理自己的服务器、节点和员工设备。
3. 服务器服务商：为客户交付可视化管理面板。
4. 跨境技术服务商：提供部署、维护、故障排查和长期升级服务。
5. 企业私有化客户：需要多服务器总控、客户后台、权限隔离和审计日志。

## 产品版本

### Community 基础版

完全开源，免费使用。

功能：

- 单机部署
- 管理员登录
- 服务器状态
- 节点管理
- 客户账号
- 订阅链接
- 流量统计
- 基础日志
- 一键安装脚本

### SaaS / Controller 总控版

同样开源，面向有技术能力的部署者。

功能：

- 多服务器管理
- Agent 心跳上报
- 多客户权限隔离
- 套餐管理
- 到期控制
- 流量统计
- 批量部署
- 批量更新
- 备份恢复
- 工单系统预留
- 代理商系统预留

## 收费点

ZXY Panel 不靠限制源码收费，而靠官方能力收费：

- 官方安装部署
- 官方稳定版构建
- 官方更新源
- 官方安全补丁
- 官方维护订阅
- 私有化部署
- 远程故障排查
- 数据迁移
- 企业培训
- 定制开发
- 品牌认证服务
